2.基础

注释

头部

可以添加在头部区域的元素标签为: <title>, <style>, <meta>, <link>, <script>, <noscript> 和 <base>。是所有头部元素的容器。

title

  1. 定义了浏览器工具栏的标题
  2. 当网页添加到收藏夹时，显示在收藏夹中的标题
  3. 显示在搜索引擎结果页面的标题

base

该标签作为HTML文档中所有的链接标签的默认链接

link

用于链接到css

4.关于插件

Chinese (Simplified)

支持中文

Auto Rename Tag

当修改一个标签的种类时，自动修改另一个对应标签的名称。

Live Server

生成动态网站，实时预览网页效果！

Prettier

每次保存时自动控制代码缩进，属于一用离不开

Image preview
